Проект Plazmatrone - софт, идеи, документация по плазматрону кафедры экспериментальной физики (Кокорин А. Ф.)
Структура файловой системы проекта
.
	/docs - документация (описание структуры, принципов работы и алгоритмов и т.д.)
	/control channel - софт для контрольного канала (проет для PIC18F452 или для другого ядра (при развитии проекта может быть выбрана
	                   разработка с использованием другого контроллера/процессора, например Cortex-M3)), C-проект;
	/measure channel - софт для измерительного канала (проет для PIC18F4550 или для другого ядра(при развитии проекта может быть выбрана
	                   разработка с использованием другого контроллера/процессора, например Cortex-M3)), C-проект;;
	/convertor - софт для ПЛИС, реализация конвертора на кристалле, Verilog-проект;
	/rules - правила работы с репозиторием (ОБЯЗАТЕЛЬНО для исполнения) и стиль кодирования проектов.

Папки добавлять тематически если требуется под контроль версий добавить, например, проект Windows-приложения для управления устройствами
необходимо создать папку с названием приложения и, соотвественно, там разместить проект.

В каждом фолдере имеет смысл держать файл readme.txt для описания набора инструментов и их версий для сборки конкретной части проекта, для
документации порядок чтения документов.
